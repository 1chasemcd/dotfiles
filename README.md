# dotfiles
The configuration files for my Arch Linux setup

## Symlinks
In order to make these configuration files work, the following symlinks must first be made:

* ln -s ~/.config/zsh/zshrc ~/.zshrc
* ln -s ~/.config/xournalpp ~/.xournalpp
